# gh-deployment-workflow

# GitHub Pages Deployment using GitHub Actions

This project demonstrates how to automatically deploy a website to GitHub Pages using GitHub Actions.

## Features

- Automatic deployment
- Trigger only when index.html changes
- Deploys to GitHub Pages

## Workflow

1. Any change to `index.html`
2. Push to the `main` branch
3. GitHub Actions workflow runs
4. Website is deployed to GitHub Pages
